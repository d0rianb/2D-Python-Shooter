### Installation
- Mac :
	 - `sudo python3 main.py` (in the `client` repository)
- Windows :
    - Execute `main.py` with `python 3`

### Commands
Keybind can be edit in the game preferences.
Default keybindings is :
- __Move__  : Z, Q, S, D (or directional arrows)
- __Shoot__ : Left Clic
- __Dash__ : Left Shift
- __Reload__ : R
- __Help__ : H
- __Melee__ : E
