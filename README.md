# 2D Python Shooter

## Instalation


- Mac :
	 - `sudo python3 main.py` (dans le dossier du jeu)
- Windows :
    - Exécuter `main.py`

## Commandes
- **Se déplacer**  : Z, Q, S, D (ou flèches directionnelles)
- **Tirer** : Clic Gauche
- **Dash** : Majuscule gauche
- **Recharger** : R
- **Help** : H
- **Dash Preview** : H
